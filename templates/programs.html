3. Synfig

E ste es un software de código abierto (Open Source) que permite al artista la creación de animaciones usando huesos o fotogramas.

Como también es de código abierto, este programa le permite a programadores poder ayudar al proyecto para solucionar errores en el programa, hacer modificaciones como la creación de plugins de herramientas para mejorar la experiencia de usuario a los artistas y animadores.

4. Pencil2D

E ste es un software de código abierto (Open Source) que permite a los artista la creación de animaciones simples.

Este software es ideal para animaciones simples o complejas, una interfaz agradable y simples y con una disposición para agregarle mejorar al proyecto en sí mismo, ya que es un software libre y de código abierto.

5. Blender

E ste es un software de modelación y animación en 3D gratuito y también de código abierto (Open Source).

Este es uno de los más populares ya sea por las herramientas que provee el programa, la calidad que tiene para la animación y modelado 3D y las modificaciones (Plugins) que se le pueden agregar al programa de forma rápida y simple para aumentar la comodidad del mismo programa o para agregar funciones nuevas al programa.
